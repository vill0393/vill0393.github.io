Five HTML tags I have learned so far

Paragraph Tag
To create a paragraph we surround the text with <p></p> tag.
Emphasis
To emphasize important text we use <em></em> tag.
Highlighting text
We can highlight a piece of text usign the <mark></mark> tag.
Cite
We use the <cite></cite> tag to indicate where the citation is from.
Adress
We use the <adress></adress> tag to add contact information, email, tel, etc.
